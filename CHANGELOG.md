# Changelog

Todas as mudanças relevantes deste projeto serão documentadas aqui.

O formato segue o padrão Keep a Changelog e versionamento semântico (SemVer).

---

## [1.0.0] - 2026-04-08

### Added
- Implementação da interface gráfica (GUI) utilizando CustomTkinter
- Integração entre GUI, CLI e sistema de persistência em JSON
- Visualização de ocorrências em formato de cards organizados por status
- Funcionalidades completas de CRUD diretamente pela interface gráfica

### Improved
- Expansão da interface CLI para suportar CRUD completo (criar, listar, atualizar, buscar e remover ocorrências)
- Integração compartilhada de dados entre CLI e GUI através de arquivo JSON
- Atualização e adaptação dos testes automatizados para suportar a nova estrutura
- Melhorias na organização do código e separação de responsabilidades
- Refinamento da experiência do usuário na aplicação

---

## [0.1.0] - 2026-04-07

### Added
- Criação da estrutura inicial do projeto
- Implementação do sistema de ocorrências em memória
- Interface CLI funcional
- Funcionalidades de criar, listar e atualizar status de ocorrências
- Testes automatizados com pytest
- Configuração de linting com Ruff
- Configuração de integração contínua com GitHub Actions (CI)

### Improved
- Organização do código e separação de responsabilidades
- Padronização da estrutura do projeto
