# Contribuindo

Obrigado por considerar contribuir com este projeto!

## 📌 Como contribuir

1. Faça um fork do repositório
2. Crie uma branch para sua feature ou correção:
   ```bash
   git checkout -b feature/minha-feature
   ``` 
3. Faça suas alterações
4. Commit suas mudanças:
   ```bash
      git commit -m "feat: adiciona nova funcionalidade"
   ```
5. Envie para o seu repositório:
    ```bash
      git push origin feature/minha-feature
   ```
6. Abra um Pull Request

## 🧪 Boas práticas
- Mantenha o código organizado e legível
- Siga o padrão de nomenclatura já existente no projeto
- Execute os testes antes de enviar:
    ```bash
  pytest
  ```
- Verifique o lint:
    ```bash
  ruff check .
  ```
  
## 💡 Sugestões de contribuição
- Melhorias na interface gráfica (GUI)
- Novas funcionalidades para o sistema
- Otimizações de código
- Correções de bugs
- Melhorias na documentação

## 📬 Contato
Em caso de dúvidas, abra uma issue no repositório.
